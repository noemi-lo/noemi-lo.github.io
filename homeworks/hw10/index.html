Take a look at the structure in the HTML file. (Don't worry about everything inside the last script tag for now.) Also, notice how all the outfits are stacked on top of drake right now.

We need to hide all the outfits and selectively pick which ones to show based on what the user clicks. Add "display: none;" into the .outfit class to hide all outfits:

.outfit {
z-index: 10;
position: absolute;
display: none;
}

Part 2: Dress Drizzy

Now, we move onto dressing Drake. There is a "script" folder that contains a script file, called "script.js". Link this JS file in your HTML.

<script src="assets/script/script.js"></script>

The script file starts with how to make the first outfit appear on Drake when you click on it. The other outfits follow the same process.

Outfit one: We're going to "hide" and "show" the body and outfits to make it seem like we're putting the selected outfit on Drake. Get the #s1 element. When we click on the #s1 element, we should hide the #body element (Hint: set the #body's display property to none).

Next, we're going to hide all the outfits. Let's get the list of .outfit elements (Hint: look into getElementsByClassName). Use a for loop to index through every outfit in this list, and hide each outfit (Hint: we have to repeat this code for every outfit... how can we shorten our code... let's make a function!).

Finally, to make it seem like Drake has changed into the outfit we selected, let's show the #o1 element (Hint: set the o1's display property to block).

Hint: Check Week 9 slides to refresh on how to get elements and Week 11 slides for the for loop syntax.

Repeat this process for the other outfits and replace s1 and o1 with the respective outfit numbers.

We're also going to add functionality to the "STRIP ME" button. Let's first get the #strip-button element. When you click on the #strip-button element, hide all the outfits (Hint: can we call the same function that we made earlier?) Then, show the body.
